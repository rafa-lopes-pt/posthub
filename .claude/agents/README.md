# Project Agents

This folder contains Claude Code agents for this project.

## Using Agents

You can use agents from the parent SPARC project or create project-specific ones.

### Using SPARC Developer Agent

In Claude Code, use:
```
/agents
```

Or directly:
```
@sparc-developer Build a new component for this project
```

### Creating Project-Specific Agents

Copy and customize agents from the SPARC project:
```
cp ../../sparc/.claude/agents/sparc-developer.md ./custom-agent.md
```

Then edit the configuration to be project-specific.
